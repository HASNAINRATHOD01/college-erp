# 🎓 College ERP System

A modern Full Stack College ERP System built using **React**, **Django REST Framework**, **SQLLite**, and **Machine Learning**. The project provides a centralized platform for managing students, teachers, attendance, marks, notices, timetables, and AI-based student performance prediction.

---

## 🚀 Project Overview

The College ERP System is designed to simplify college administration by providing secure role-based access for **Admin**, **Teacher**, and **Student** users.

It also includes an **AI Prediction Module** that analyzes attendance and academic performance to predict student grades and identify at-risk students.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/HASNAINRATHOD01/college-erp.git
```

```
cd college-erp
```

---

## Frontend

```
cd frontend
npm install
npm run dev
```

---

## Backend

```
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
