# ЁЯОУ College ERP System

A modern Full Stack College ERP System built using **React**, **Django REST Framework**, **PostgreSQL**, and **Machine Learning**. The project provides a centralized platform for managing students, teachers, attendance, marks, notices, timetables, and AI-based student performance prediction.

---

## ЁЯЪА Project Overview

The College ERP System is designed to simplify college administration by providing secure role-based access for **Admin**, **Teacher**, and **Student** users.

It also includes an **AI Prediction Module** that analyzes attendance and academic performance to predict student grades and identify at-risk students.
